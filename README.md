# ZALLSTORE DIGITAL HUB

Aplikasi fullstack modern bertema cyber-premium untuk marketplace akun premium serta jasa videografi & fotografi.

## Stack
- React + Vite + TailwindCSS + Framer Motion
- Node.js + Express
- Firebase / Firestore ready dengan fallback in-memory
- JWT auth sederhana

## Menjalankan project
```bash
npm install
npm run dev
```

Frontend tersedia di `http://localhost:5173` dan backend di `http://localhost:3001`.

## Struktur
- `src/` frontend SPA premium mobile-first
- `server/` backend Express REST API
- `.env.example` template konfigurasi Firebase dan JWT
